Knowledge Grapht — adaptive learning platform

I designed and built a classroom-ready study platform to generate adaptive questions from course materials. The goal is to reduce cognitive overhead and help students retain information through spaced repetition.

Project overview

Role: Designer and Full-stack developer

Year: 2024 to Present

Stack: Next.js, React, Supabase, Vercel

What I built

  • Developed two connected applications: a React Native mobile app for students and a React + TypeScript web dashboard for instructors.
  • Designed and implemented the core knowledge graph structure, allowing students to browse class topics as a visual hierarchy instead of linear lists.
  • Built custom question-generation flows using LLM prompt templates that adapt to the student’s current position in the graph.
  • Created lesson, topic, and class management tools for instructors, with real-time updates powered by Supabase row-level security and live subscriptions.
  • Led iterative usability testing with students and faculty to refine navigation patterns and reduce cognitive friction during studying.
